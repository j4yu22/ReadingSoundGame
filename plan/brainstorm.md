Funamental game aspects
No letters, only tokens; auditory sounds/words
One token per phoneme in a word
Student says the sound of the phoneme as they drag the token down to the bottom of the screen
For the sake of the dialogue, 'Arthur' is the ai voice the students will be interacting with

Student is correct example:
    Arthur: "Say 'and'"
    Student: "and"
    Arthur: "Now pull down a token for each sound in 'and'"
    3 gray unlabeled tokens appear
    The student touches the first token, and while dragging it down says: "ah"
    The token turns green
    The student touches the second token, and while dragging it down says: "nnn"
    The second token turns green
    The student touches the third token, and while dragging it down says: "duh"
    The third token turns green
    Arthur: "Now say 'and' but dont say 'duh'"
    The third green token vanishes
    The student says: "an"
    END OF EXERCISE

If student makes a mistake, prompt ask the student to follow along, and have Arthur voice the whole thing out.

Design
A big tower with 20 floors
Each floor has combats, when the user correctly says the word they will progress to another, if they dont then they will take damage
The run is over when the users hp hits zero, or after they have slain x number of enemies in a row
Each enemy that is slain will drop money that the user can spend in a shop while they are not in runs
Not exactly sure on what items would be balanced, but definitely cosmetics and stuff would be cool
Daily streaks with rewards
After a floor has been cleared, it will unlock endless mode for that floor
Can find loot in between combats that can be brought back if a run is successful

Scoring
At the beginning there is an assessment to see which floor the user should be in
The assessment will test 3 words starting at level 1, and moving up until the user fails a word
They are placed on the floor at which they failed
Each floor has “combats” where the user attempts words
If the user correctly says the word within 2 seconds of the prompt, the enemy is slain and another appears
If the user says the correct word but takes more than 2 seconds they trade with the enemy
If the user says the incorrect word their attack is parried and riposted
The only way to kill an enemy is through the first option
The user has a set amount of hp
The number of combats is arbitrary, but it should continue until the user has slain x enemies in a row
Criteria for a floor being cleared is unknown right now, but it's probably after a few successful adventures
After the floor is cleared the user will ascend to the next

Loot Ideas
Health potion - restores health, consumable
Shield - protects the user from x trades before it breaks
Armor - protects the user from x trades or ripostes before it breaks
Dagger - slay x enemies without needing to say the word (acts as a skip)
Backdoor Key - allows the user to return home part way through a run, keeping their items
